## How to use ?
โคลนโปรเจคนี้ไปเลยยค้าบวัยรุ่นนนนน

``` git clone https://github.com/SassyxD/engdis-hax ```


แล้วก็ลง package ทั้งหมดก่อน
โดยใช้คำสั่ง

``` npm install ```

หลังจากลง package เสร็จแล้วสามารถใช้งานได้โดยใช้คำสั่ง

```js
node .
```

สามารถใช้งานได้แล้ววววววว !!!
